Reference Code Syntax:

    VARIABLES: 
        Syntax -    [all lower case]
        EXAMPLES -  [var1, api_key]

    GLOBAL VARIABLES: 
        Syntax -    [include _ infront]
        EXAMPLES -  [self._global_storage, self._apikey]

    CLASS:
        Syntax -    [Put Py infront]
        EXAMPLES -  [Pyclass,  PyCT]

    FUNCTIONS:
        Syntax -    [all lower case, prefix is class name]
        EXAMPLES -  [Pyclass_generic_function_name, CT_Query_Server_Status]
